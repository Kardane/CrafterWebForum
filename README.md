# CrafterForumWeb (Windows Native Setup)

This repository is now standardized for Windows PowerShell local development

## 1) Prerequisites

- Windows 10/11
- PowerShell 5.1+ or PowerShell 7+
- Node.js 20+ recommended
- npm 10+ recommended

Check versions

```powershell
node -v
npm -v
```

## 2) Quick start

Run at repository root

```powershell
npm run setup:win
```

This script performs

- dependency install
- `.env.local` bootstrap from `.env.example` when missing
- automatic `NEXTAUTH_SECRET` generation when placeholder is found
- Prisma generate and db push
- Playwright Chromium install
- local environment doctor checks

## 3) Required environment values

After `setup:win`, verify `.env.local` values

- `NEXTAUTH_SECRET`: auto-generated by setup script, replace if needed
- `DATABASE_URL`: local default `file:./dev.db` is valid

Generate a secret

```powershell
node -e "console.log(require('node:crypto').randomBytes(32).toString('hex'))"
```

## 4) Daily commands

npm run build
